# Udemy-website
My project is a simple front-end clone of the Udemy online learning platform, created using **HTML** and **CSS**. The website focuses on a clean user interface and an organized layout to showcase different courses in an easy and visually appealing way. It highlights basic web design and layout structuring skills.

---

###  Features

* Homepage with featured courses
* Course cards arranged in a grid layout
* Clean and modern UI design
* Fully responsive using CSS
* Simple and easy-to-navigate structure
